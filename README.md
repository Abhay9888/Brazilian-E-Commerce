# 🛒 E-Commerce Analytics Dashboard | Power BI + PostgreSQL

##  Project Overview
This project is a complete **E-Commerce Analytics Dashboard** developed using **Power BI** and **PostgreSQL** to analyze business performance across multiple domains including:

- Revenue Analytics
- Customer Analytics
- Seller Performance
- Delivery Operations
- Payment Insights

The dashboard provides interactive business intelligence with advanced KPIs, dynamic filtering, drill-through analysis, and time-intelligence calculations.
